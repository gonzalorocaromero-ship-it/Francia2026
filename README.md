# Francia 2026 · App Familiar 🗼

Guía de viaje interactiva para París y Disneyland Paris.
**29 Agosto — 4 Septiembre 2026 · Le Victoria Garden, Serris**

---

## 🚀 Publicar en GitHub Pages (5 minutos)

### Paso 1 — Crear repositorio
1. Ve a **github.com** e inicia sesión (o crea cuenta gratis)
2. Pulsa el **+** arriba a la derecha → **New repository**
3. Nombre: `francia2026` (o el que quieras)
4. Márcalo como **Public**
5. Pulsa **Create repository**

### Paso 2 — Subir los archivos
1. En la página del repositorio, pulsa **uploading an existing file**
2. Arrastra estos 5 archivos de golpe:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Pulsa **Commit changes**

### Paso 3 — Activar GitHub Pages
1. Ve a **Settings** (pestaña del repositorio)
2. En el menú izquierdo: **Pages**
3. En "Source" selecciona: **Deploy from a branch**
4. Branch: **main** / Folder: **/ (root)**
5. Pulsa **Save**

### Paso 4 — Tu URL (lista en ~2 minutos)
```
https://TU_USUARIO.github.io/francia2026/
```

### Paso 5 — Instalar como app
1. Abre la URL en **Chrome para Android**
2. Aparece el banner "Instalar Francia 2026" → pulsa **Instalar**
3. El icono 🗼 aparece en la pantalla de inicio
